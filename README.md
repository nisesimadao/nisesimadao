# Nisesimadao

趣味でソフトウェア開発･映像制作･作曲活動を行っています。  
macOS / iOS を中心に、デスクトップアプリやユーティリティツールの制作をしています。

## Focus
- Swift / SwiftUI を用いた macOS・iOS アプリ開発
- 小規模で実用的なツールの設計・実装
- クロスプラットフォームを意識した開発

## Main Repositories
### SimpleWakeOnLan
SwiftUI 製の Wake on LAN ユーティリティ（macOS / iOS）。

- 複数デバイスの登録・管理
- WoL パケット送信
- Ping によるオンライン状態確認
- SwiftUI によるシンプルな UI 構成

### TimeTree Desktop
Electron ベースのデスクトップアプリ。

- Web サービスをデスクトップ向けに最適化
- カスタムタイトルバー
- ダークモード対応

### SpeakHand
Windows 向けオーディオデバイス切り替えツール。

- スピーカー／出力デバイスの即時切り替え
- C# / .NET による実装

### File2Pixel
ファイルを画像データとして変換するユーティリティ。

- 任意ファイルをピクセル画像へ変換
- 画像から元ファイルの復元が可能
- C# による Windows 向けツール
